# ALX HTML & CSS Advanced Project

## Overview

This project is part of the **HTML, Advanced Project** from ALX.  
The main focus of this project is **CSS styling and layout** of web pages based on a Figma design.  

You will find the HTML structure already provided, and the CSS is applied to achieve a responsive, visually appealing layout.

---

## Files in this Repository

**Directory:** `css_advanced`

- `index.html` – The main HTML file for the project.  
- `README.md` – This documentation file.

---

## Objectives

1. Implement the CSS design provided in the Figma file.  
2. Ensure the web pages are responsive and adapt to different screen sizes.  
3. Apply advanced styling techniques such as Flexbox, custom fonts, colors, and layout adjustments.  
4. Integrate images, logos, and any assets required by the design.  
5. Maintain semantic HTML structure while enhancing the visual appearance.

## Instructions

1. Open `index.html` in a web browser to view the page.  
2. The page layout follows the Figma design closely.  
3. Make sure all images and logos are linked correctly in the HTML.  
4. CSS styling should reflect spacing, colors, typography, and responsive behavior as outlined in the design.

---

## Notes

- Some CSS values may be floats; rounding them is acceptable.  
- Ensure the website is tested on multiple screen sizes for responsiveness.  
- Maintain semantic HTML for accessibility and clarity.
